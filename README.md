**UK Horse Race Predictor 🏇**

*Developer*: @el_bitzo | *Discord*: @hustlekiddo

After searching endlessly for a free, reliable AI analyzer for UK horse racing and finding absolutely nothing, I decided to build my own. UK_Predictor is an automated machine learning pipeline that bypasses Cloudflare, scrapes live UK horse racing data, and uses an XGBoost AI model to predict high-probability winners.

**Features:**
Cloudflare Bypass: Utilizes undetected-chromedriver to securely navigate bot-protection and scrape live daily UK/IRE racecards.
Smart Data Merging: Cleans, formats, and matches live horse names directly to a database of 25,000+ historical races.
XGBoost AI Engine: Generates win probabilities based on 12 key features (Jockey Win Rate, Horse Win Rate, RPR, Weight Diff, etc.).

**Installation:**
**Clone the repository:**
git clone https://github.com/el_bitzo/UK-Predictor.git

cd UK-Predictor

**Install the required Python libraries:**
pip install -r requirements.txt

Ensure Google Chrome is installed: The web scraper requires a standard, up-to-date installation of Google Chrome on your machine to operate.

**How To Use**

The Easy Way (One-Click)
Run the automated pipeline orchestrator. This will safely open Chrome, scrape today's races, cross-reference the historical database, and instantly print the AI's Top 3 predictions:
**python auto_run.py**

**The Manual Way**

If you prefer to run the pipeline steps individually:

**Scrape Today's Races:**

python fetch_real.py

**Train the AI Model (Run this at least once):**

python main.py --retrain

**Generate Predictions:**

python main.py --predict-only

**Daily Maintenance (Keeping the AI Smart)**
The AI relies on historical data to make accurate predictions. As races finish, you need to add the official results to your database so the AI learns from them.

**Run this command every morning before you pull today's predictions:**

python update_database.py

**Once the database is updated with yesterday's results, retrain the AI's brain so it learns the new patterns:**

python main.py --retrain

**Backtesting & PnL**

To see the mathematical edge of the AI, you can run a Time-Series Cross-Validation over the historical database to simulate ROI:
**python pnl_chart_ev.py**
